BMAD Breakthrough Method for Agile AI-Driven Development

A comprehensive framework that integrates agile methodologies with AI agents to streamline software development through structured workflows and specialized roles.

01

Two Core Phases

The Foundation of BMAD

Phase A

Agentic Planning

Structured creation of Product Requirements Documents (PRDs), system architecture, and UX briefs. AI agents collaborate with human oversight to ensure comprehensive planning before any code is written.

Environment: Web UI / Powerful IDE Agents
Phase B

Context-Engineered Development

The Scrum Master converts artifacts into detailed stories. Developers implement code while QA agents conduct structured reviews. Emphasizes lean context and gated workflows.

Environment: IDE with Integrated Agents
