Installation

One Command Away

Fastest Way
One Command. That's It.

With uvx, you can run Porterminal instantly without any installation or setup:

uvx ptn

Don't have uvx? Here's the traditional setup:

1

Clone the Repository

Start by cloning Porterminal from GitHub:

git clone https://github.com/lyehe/porterminal.git && cd porterminal
2

Install & Run

Install dependencies and start the server:

pip install -r requirements.txt && python app.py
3

Access via Browser

Open your browser and navigate to the local server:

http://localhost:5000
