Installation

Three Simple Steps

1

Add the Marketplace

Inside a Claude Code instance, add the plugin marketplace:

/plugin marketplace add jarrodwatts/claude-hud
2

Install the Plugin

Install claude-hud from the marketplace:

/plugin install claude-hud
3

Configure the Statusline

Run the setup command to configure the HUD:

/claude-hud:setup
Ready to Use
No Restart Required

The HUD appears immediately after setup ‚Äî no restart needed. It updates every ~300ms with native token data from Claude Code, not estimates.

How It Works

Under the Hood

‚Üí

Native Statusline API

Claude HUD uses Claude Code's native statusline API. No separate window, no tmux required, works in any terminal. The HUD is always visible below your input.

‚ö°

Real Token Data

Context usage comes directly from Claude Code ‚Äî not estimated. The plugin parses the transcript for tool/agent activity and updates every ~300ms.

{}

JSON Streaming

Claude Code sends stdin JSON to claude-hud, which processes it and outputs to stdout for display in your terminal. Clean, efficient, lightweight.

üìÑ

Transcript Parsing

The plugin reads the transcript JSONL file to track tools, agents, and todos. This gives you accurate, real-time information about your session.

Data Flow
Claude Code ‚Üí stdin JSON ‚Üí claude-hud ‚Üí stdout ‚Üí displayed in terminal
           ‚Üò transcript JSONL (tools, agents, todos)
