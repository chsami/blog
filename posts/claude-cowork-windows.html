Claude Cowork Is Finally on WindowsEight months of watching from the sidelines. Worth the wait?

I've been watching Mac users play with Claude Cowork since January while sitting here on Windows waiting. Today the update finally dropped. Spent the afternoon with it and wanted to write up what it's actually like before the hype cycle kicks in.

01
The waiting was genuinely frustrating
Platform envy is real
Watching Mac users get all the good stuff

When Cowork launched the demos were all over my feed. Mac users sorting hundreds of files in 30 seconds, turning receipt photos into expense spreadsheets, building documents out of a folder full of scattered notes. It looked useful in a real way, not just in a controlled demo kind of way. The frustrating part wasn't just missing out. It was that the thing clearly worked and I just couldn't use it because of which OS I'm on.

The announcement landed this morning

Anthropic pushed the update to Claude Desktop for Windows this morning. The changelog was short: "Cowork is now available on Windows, powered by WSL2." No blog post, no announcement thread, just a quiet update notification in the app. I already had WSL2 enabled from some old dev work so setup took about 10 seconds. If you've never touched WSL2 before there's one extra step, I'll cover that below.

02
First thing I actually tried
No carefully chosen demo folder
I pointed it at a real mess

I have a folder called "work-misc" that's been collecting random files for two years. Meeting notes saved as screenshots, half-finished drafts, PDFs I downloaded and never opened, zip files I was "going to deal with later." 340 files. No structure at all. I pointed Cowork at it and asked it to make sense of everything.

Step 1
Claude proposed a folder structure before touching anything. It showed me what it planned to create and where things would go. I could read through the whole plan and approve or edit before a single file moved.
Step 2
It ran. 340 files sorted into 12 folders in under two minutes. The groupings made sense. Client documents separate from internal notes, invoices separate from articles, screenshots grouped by topic. I didn't have to correct anything.
Step 3
After it finished it gave me a short summary of what it did. Which categories it created, how many files went where, and three files it wasn't sure about that I should check myself.
The thing that actually surprised me

Those three flagged files were genuinely ambiguous. It got that part right too. Everything else it handled without asking. That's the bit I didn't expect. I figured it would either be overconfident and make bad calls, or keep stopping to ask me what to do. It did neither.

03
The WSL2 requirement
What you need before you start
Why WSL2?

The Mac version uses Apple's Virtualization Framework to spin up a sandboxed Linux VM. On Windows, Anthropic went with WSL2 as the equivalent. The end result is the same: Claude runs inside an isolated Linux environment that can only see the folder you give it. It has no access to the rest of your drive, other apps, or anything else on your system.

Setup if you don't have WSL2

If WSL2 isn't enabled, Claude Desktop will walk you through it. Open PowerShell as administrator, run wsl --install, restart. Takes about five minutes total. After that you don't touch it again. Cowork just works from the app like everything else.

What WSL2 means for you in practice
Security
The sandbox is real. Claude can't see anything outside the folder you mount. Give it your Documents folder and that's all it gets. This is enforced at the VM level, not just a setting in the app.
Performance
File operations through WSL2 on Windows paths are a bit slower than native. I didn't notice it with 340 files, but if you throw a huge deeply nested project at it, expect it to take slightly longer than it would on a Mac.
Stability
No crashes in my afternoon of use. Two sessions, both completed cleanly. WSL2 has been around long enough that it's solid at this point.

How to get started
Five steps from zero to running
Requirements
1Windows 10 version 2004 or later (or Windows 11)

3Claude Desktop for Windows, updated to the latest version
4A paid Claude subscription (Pro, Max, Team, or Enterprise)

1Open Claude Desktop and switch to the Cowork tab

05
Compared to the Mac version
Functionally identical, minor differences
Same
The core experience

Picking a folder, describing a task, reviewing a plan, watching it run. All identical. Same interface, same behavior, same model.

Same
Plugin support

The plugins from January work on Windows too. Sales, finance, legal workflow extensions, all there. Nothing Mac-exclusive that I could find.

Different
Setup prerequisite

On Mac, Apple's VM framework is built into the OS so setup is instant. On Windows you need WSL2 enabled first. It's a one-time thing and takes five minutes, but it's still an extra step Mac users don't have to think about.

06
Tasks worth trying first
Where it actually earns its keep
File cleanup and organization

This is the obvious starting point and it delivers. Point it at any chaotic folder and ask it to sort things. Downloads, old project archives, a desktop that's become a dumping ground. It reads file contents, not just names and extensions, so it can actually tell the difference between an invoice PDF and a whitepaper PDF.

Synthesizing notes into documents

Give it a folder of messy notes, bullet points saved as text files, random half-formed ideas, and ask it to build something coherent out of it. I tried this with a month of meeting notes. It came back with something I'd actually send to someone. Organized by topic, readable, not just a dump of everything I wrote.

Extracting data from documents

Receipt screenshots, scanned invoices, PDFs with tables. Cowork pulls the data out and builds a spreadsheet. I tested it with a mix of receipt photos and PDF invoices and it correctly grabbed vendor, date, and amount from all of them, including photos taken in bad lighting.

Renaming and restructuring projects

If you have a project folder that grew without a plan, inconsistent naming, files in the wrong places, duplicates everywhere, just describe the structure you want and let Cowork sort it out. The kind of thing that takes 20 minutes of careful manual renaming gets done in under a minute.

Bottom line
It was worth the wait. Go try it.

Cowork on Windows isn't a cut-down version of the Mac experience. It's the same thing. If you've had a Claude subscription this whole time and were waiting for this, go open the Cowork tab and pick the messiest folder on your machine.
