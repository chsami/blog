Ralph Autonomous AI Development Loop for Claude Code

Ralph is a bash-based automation wrapper exclusively for Claude Code, Anthropic's official CLI tool. It creates a continuous development loop where Claude Code iteratively builds your project from a prompt file, analyzing each response to determine when work is complete. Not for Cursor, Copilot, or other AI tools. This is pure Claude Code automation.

The Loop Concept

Ralph wraps Claude Code in a while loop, repeatedly invoking it with your PROMPT.md file. After each execution, Ralph analyzes Claude's response to determine if the task is complete or if another iteration is needed. This continues until project objectives are met.

üîß

Prerequisites

Requires: Claude Code CLI (claude command), tmux for session management, and bash. Ralph orchestrates these tools together. It doesn't replace Claude Code, it automates the repeated invocation pattern that complex projects require.

    # Invoke Claude Code CLI with your prompt

Intelligent Exit Detection

How Ralph Knows When to Stop

üß†

Response Analyzer

After each Claude Code execution, Ralph's response analyzer parses the output using semantic pattern matching. It looks for completion signals like "all tasks complete", "no remaining work", or explicit done markers in Claude's response.

üìã

Task File Monitoring

Ralph monitors @fix_plan.md for remaining tasks. When Claude marks all items complete or the file indicates no pending work, Ralph triggers exit. This file acts as the shared state between Ralph and Claude Code.

üõë

Exit Thresholds

Configurable thresholds prevent premature or delayed exits. Ralph tracks consecutive "complete" signals, requiring multiple confirmations before stopping to avoid false positives from ambiguous responses.

‚è±

Rate Limit Detection

Ralph detects Claude's API rate limits and 5-hour usage cap. When hit, it displays a countdown timer and offers options: wait for the limit to reset or gracefully exit the loop.

üîÑ

Circuit Breaker

After repeated failures or stuck states, the circuit breaker trips. This prevents infinite loops when Claude Code encounters errors, unresolvable conflicts, or repeatedly produces the same output.

üìä

Max Iterations

Hard limit on loop iterations (default: 100 calls/hour). Even if exit detection fails, Ralph will stop at the configured maximum to prevent runaway API usage and costs.

Getting Started

Installation & First Run

1

Install Claude Code

First, install Anthropic's Claude Code CLI. Visit claude.ai and follow the installation guide. Authenticate with claude login. Ralph cannot work without Claude Code installed.

2

Install Ralph

Clone the repository and run the installer. Ralph installs globally, adding ralph, ralph-setup, and ralph-import commands to your PATH.

3

Create PROMPT.md

Write your project requirements in PROMPT.md. This is what Claude Code will work from. Be specific about features, tech stack, and completion criteria.

Full Installation Steps
# 1. Prerequisites: Install Claude Code CLI (required!)
# Follow instructions at: https://claude.ai/code
claude --version        # Verify it's installed
claude login           # Authenticate with Anthropic

# 2. Install Ralph
git clone https://github.com/frankbria/ralph-claude-code.git
cd ralph-claude-code
./install.sh

# 3. Create a new project
ralph-setup my-project
cd my-project

# 4. Edit PROMPT.md with your requirements
vim PROMPT.md

# 5. Start the autonomous loop
ralph --monitor --verbose

CLI Reference

Command Line Options

Ralph Command Options
ralph [OPTIONS]

  -h, --help          Show help message
  -c, --calls NUM     Set max calls per hour (default: 100)
  -p, --prompt FILE   Set prompt file (default: PROMPT.md)
  -s, --status        Show current status and exit
  -m, --monitor       Start with tmux session and live monitor
  -v, --verbose       Show detailed progress updates
  -t, --timeout MIN   Set Claude Code timeout (1-120 minutes)

# Project setup commands
ralph-setup project-name     # Create new Ralph project
ralph-import prd.md project  # Convert PRD to Ralph project
ralph-monitor              # Manual monitoring dashboard
